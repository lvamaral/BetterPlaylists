# API Endpoints

## HTML API

### Root
	GET / - loads React web app

## JSON API

### Users
	POST /api/users

### Session
	POST /api/session
	DELETE /api/session

### Playlists
	GET /api/playlists
	Playlist index

	POST /api/playists
	GET /api/playlists/:id
	PATCH /api/playlists/:id
	DELETE /api/playlists/:id

### Songs
	GET /api/songs/:id
